📝 Keeper App

A simple and elegant note-taking web application built using React.
Inspired by Google Keep, this app allows users to quickly create, view, and delete notes in a minimal and user-friendly interface.

🚀 Features
✍️ Create and save notes instantly
🗑️ Delete notes easily
⚡ Fast and responsive UI
🎯 Clean and minimal design
📱 Works on different screen sizes
🛠️ Tech Stack
⚛️ React (Functional Components & Hooks)
🧩 JavaScript (ES6+)
🎨 HTML5 & CSS3
🌐 Live Demo

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/vaibhavb-cmd/Keeper.git
cd Keeper

Install dependencies:

npm install

Run locally:

npm start
📦 Deployment

This project is deployed using GitHub Pages.

npm run deploy
